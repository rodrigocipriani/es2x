# es2x - Ecma Script 2xxx 
Javascript Helpers, Tools, Components, etc.

## How to use

Run `npm i --save es2x`

## How to develop

- Clone this repo
- Run `npm i`
- To Build: `npm run build`
- To run with watcher: `npm start`
